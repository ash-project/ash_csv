# AshCsv

A CSV data layer for Ash. Currently only meant for demos. The code is messy, its not made to scale, and it doesn't have code for handling most primitive types.

## Installation

```elixir
def deps do
  [
    {:ash_csv, "~> 0.7.0-rc0"}
  ]
end
```
